# Landing Page Conversion Skill

Use this skill to review or create landing pages that clearly explain the offer, reduce hesitation, and drive measurable action.

## Core Principle

Traffic is not the win. A landing page wins when the right visitor understands the offer fast, trusts it enough, and knows exactly what to do next.

## Load Triggers

Load this skill when the task mentions:

- landing page, homepage, sales page, pricing page
- conversion, CTA, offer, lead generation, sign-up, checkout
- hero section, copy, trust proof, objection handling
- improve page performance from traffic, ads, SEO, or AEO

## Conversion Audit

### 1. Above The Fold

Check:

- Clear offer in one glance
- Specific audience
- Primary CTA visible
- Visual supports the offer
- No generic slogan replacing the actual value

Output:

```md
Hero verdict:
Headline:
Subheadline:
Primary CTA:
Secondary CTA:
Missing clarity:
```

### 2. Offer Clarity

```md
Who it is for:
Problem it solves:
Outcome promised:
Mechanism:
Time to value:
Proof:
Risk reversal:
```

### 3. Trust & Proof

Use the strongest available proof:

- customer result
- testimonial
- case study
- screenshot
- number or benchmark
- credential
- guarantee
- demo

Rule: if there is no proof, do not make a big claim.

### 4. Objection Handling

Map objections:

```md
Too expensive:
Too hard:
Too risky:
Not for me:
No urgency:
Already have alternative:
```

### 5. CTA System

Every page needs:

- one primary CTA
- consistent CTA wording
- CTA repeated after proof
- low-friction alternative when commitment is high

## Brutal Rules

- If the visitor cannot explain the offer in 5 seconds, rewrite the hero.
- If CTA text is vague, replace it.
- If the page asks for trust before showing proof, reorder it.
- If there are multiple competing CTAs, pick one primary action.
- If pricing is hidden but price anxiety is obvious, address it.
